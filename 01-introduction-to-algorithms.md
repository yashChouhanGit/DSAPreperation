
Title: Introduction to Algorithms

Q1: What is an Algorithm?

Answer:
An algorithm is a finite set of well-defined step-by-step instructions used to solve a specific problem or perform a computation.

Example:
1. Start
2. Input two numbers A and B
3. Compute C = A + B
4. Display C
5. Stop


Q2: Why do we write algorithms?

Answer:
Algorithms are written to define a clear logical procedure for solving a problem before implementing it in a programming language. They help developers understand the solution, analyze complexity, and build efficient programs.


Q3: Who writes algorithms?

Answer:
Algorithms are written by programmers, software engineers, and computer scientists. They design algorithms to solve computational problems before writing the actual program.


Q4: When do we write algorithms?

Answer:
Algorithms are written before coding. They act as a blueprint for the program.

Typical development process:
Problem → Algorithm → Program → Output


Q5: What are the characteristics of a good algorithm?

Answer:

1. Input
An algorithm should accept zero or more inputs.

2. Output
It must produce at least one output.

3. Definiteness
Each step must be clear and unambiguous.

4. Finiteness
The algorithm must terminate after a finite number of steps.

5. Effectiveness
Each instruction must be basic and executable.


Q6: What is the difference between an Algorithm and a Program?

Algorithm:
A step-by-step logical procedure to solve a problem. It is language independent.

Program:
A program is the implementation of an algorithm written in a specific programming language.

Q7: Why are algorithms important?

Answer:
Algorithms are important because they help in solving problems efficiently, optimizing performance, reducing time complexity, and improving the overall design of software solutions.
